# Color pickers

Multiple color picker controls built on Flutter's
[`flutter_colorpicker`](https://pub.dev/packages/flutter_colorpicker) package.

## Usage

Add `flet-color-pickers` to your project dependencies:

<Tabs groupId="uv--pip">
<TabItem value="uv" label="uv">
```bash
uv add flet-color-pickers
```

</TabItem>
<TabItem value="pip" label="pip">
```bash
pip install flet-color-pickers  # (1)!
```

1. After this, you will have to manually add this package to your `requirements.txt` or `pyproject.toml`.
</TabItem>
</Tabs>
## Available color pickers

- [ColorPicker](colorpicker.md)
- [BlockPicker](blockpicker.md)
- [HueRingPicker](hueringpicker.md)
